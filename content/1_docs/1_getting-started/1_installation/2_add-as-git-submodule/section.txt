Title: Installation as Git submodule

----

Abstract: 

----

Text:

Run the following command on your website’s root and Merx and it’s dependencies is installed at `site/plugins/merx`.

```bash
git submodule add https://github.com/wagnerwagner/merx.git site/plugins/merx
```

Read more about Kirby and Git on [getkirby.com](https://getkirby.com/docs/cookbook/setup/git)
